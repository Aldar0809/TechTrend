Пояснительная записка к проекту TechTrend
1. Суть проекта
TechTrend - это интернет-магазин, созданный на языке Python с использованием фреймворка Flask, специализирующийся на продаже инновационных гаджетов, аксессуаров и электроники. Проект ориентирован на пользователей, увлеченных новейшими технологиями и стремящихся быть в курсе последних трендов.
2. Функциональность
Каталог товаров: Пользователи могут просматривать широкий ассортимент продуктов, разделенных по категориям и брендам, с возможностью фильтрации и поиска.
Корзина и оформление заказа: Позволяет добавлять товары в корзину, удалять их и оформлять заказы с указанием информации о доставке и оплате.
Личный кабинет: Зарегистрированные пользователи имеют доступ к личному кабинету, где они могут просматривать историю заказов, управлять информацией профиля и оставлять отзывы о продуктах.
Администрирование: Предоставляет администраторам возможность добавлять, редактировать и удалять товары, управлять категориями и брендами.
3. Структура репозитория
/app.py: Точка входа приложения, запускающая сервер Flask и создающая базу данных.
/app/: Основной каталог с исходным кодом приложения, разделенный на подкаталоги:
/__init__.py: Инициализирует приложение и подключает необходимые модули.
/extensions.py: Определяет расширения Flask, такие как SQLAlchemy для работы с базой данных и Flask-Login для аутентификации пользователей.
/models.py: Описывает модели данных для пользователей, продуктов, категорий, брендов, заказов и отзывов.
/routes.py: Определяет маршруты приложения и обрабатывает запросы пользователей.
/static/: Содержит статические файлы, такие как CSS-стили для оформления интерфейса.
/templates/: Содержит HTML-шаблоны для отображения страниц приложения.
/instance/: Каталог для хранения файлов конфигурации и базы данных.
/requirements.txt: Список зависимостей проекта, устанавливаемых с помощью pip.
/README.md: Файл с описанием проекта и инструкциями по его запуску.
/description.txt: Дополнительный текстовый файл с описанием проекта.
4. Технологии и зависимости
Flask: Веб-фреймворк Python для создания веб-приложений.
Flask-SQLAlchemy: Расширение Flask для работы с реляционными базами данных.
Flask-Login: Расширение Flask для аутентификации пользователей.
Werkzeug: Набор утилит для веб-разработки на Python.
SQLite: Реляционная база данных, используемая для хранения данных приложения.
Requests: Библиотека Python для выполнения HTTP-запросов.
